# AI Smart Elder Care

## Developed by
Ankur Dhole
BTech IT – WIT Solapur

## Problem
Elderly people face difficulty in managing health, reminders, emergency and family communication.

## Solution
AI Smart Elder Care is a voice-enabled web application that helps elders with:

- AI Chatbot
- Health Check
- Emergency WhatsApp Alert
- Ambulance Call
- Family Dashboard
- Voice Assistant
- Reminders
- Live Location
- Helpers near Solapur

## Technology Used
HTML
CSS
JavaScript
Tailwind
Groq AI API
Weather API
Web Speech API

## Features
Voice assistant
Emergency alert
WhatsApp integration
Real-time health advice
Family communication
